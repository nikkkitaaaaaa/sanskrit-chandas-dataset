# Sanskrit Chandas Dataset

This repository contains the **dataset and resources for the FY project competition** at  
**PES Modern College of Engineering, Pune (Autonomous)**.

The project is related to **Indian Knowledge Systems (IKS)** and focuses on detecting  
**Laghu (L) and Guru (G) syllable patterns** in Sanskrit shlokas written in Devanagari.

## Contents

dataset/
- gayatri_mantra.txt
- totakashtakam.txt
- bhavani_ashtakam.txt

rules/
- laghu_guru_rules.txt

metadata/
- metre_specifications.xlsx

## Goal

Students must write a program that reads a shloka, detects the **Laghu–Guru pattern**,  
and outputs the result in a `.txt` file.

## Note

All text files are in **UTF-8 Devanagari encoding**.
